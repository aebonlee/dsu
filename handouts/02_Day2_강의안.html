[요청] 이 평가를 비판형 서술 + 포트폴리오 조합으로 바꾼다면 어떤 모습이 될지 설계해줘.

[조건] 수행 관찰과 판정은 교수자의 몫임을 전제로, AI는 기준 설계까지만.

  • 사고 동사 고르기: 요약하라(이해) < 비교하라(분석) < 평가하라(평가) < 제안하라(창안)

  • 모범답안 3종의 쓸모: 채점 일관성 확보 + 학생에게 수준별 예시 제공
  • 보안 원칙 복습: 실제 시험 문항은 유사 연습 문항으로 바꿔 실습합니다

실습 23수준 모범답안 생성
[역할] 너는 채점 기준을 답안으로 보여주는 평가 전문가야.

[요청] 1번 문항의 모범답안을 세 수준으로 써줘. (1) 우수(만점) (2) 보통(부분 점수) (3) 미흡.
[형식] 각 답안 아래 "이 답안이 이 수준인 이유"를 채점자 관점에서 2줄씩.
[예시] 이유 서술 예: "보통: 주장은 파악했으나 근거가 1개뿐이고 반론 인식이 없음".
[조건] 세 답안의 분량은 비슷하게 — 길이가 아니라 사고의 질로 수준이 갈리게.

[요청] 1번 문항을 (1) 한 단계 쉬운 버전(1학년용) (2) 한 단계 어려운 버전(3~4학년 심화용)으로 변형해줘.
[형식] "버전 / 문항 / 무엇을 바꿨나(사고 동사·조건·자료)" 3열 표.
[예시] 조절 원리 예: "쉬운 버전 — '비교하시오'를 '차이점 2가지를 찾으시오'로, 자료 제시".
[조건] 난이도가 달라진 이유를 원리로 설명해서, 내가 다른 문항에도 응용할 수 있게.

주요 내용
  • 개인화 피드백 전략: 정성 들인 표본 1개 작성 → 구조를 템플릿으로 추출 → 학생별 메모만 바꿔 적용
  • 피드백 3단 구조: 잘한 점(구체적 인용) → 성장 포인트(방법 포함) → 다음 단계 격려
  • 개인정보 원칙 복습: 학생 이름·학번 제거, 과제 내용도 최소한만 입력
  • 가짜 인용(환각 인용) 잡기: AI가 만든 참고문헌은 실존 여부를 반드시 확인 — 제목·저자·연도 검색
  • 웹 검색 활용 근거 수집: 평가 문항·피드백에 쓸 최신 사례·통계를 웹 검색으로 모으고 출처를 함께 기록
  • 검증 루틴 5단계: ①모든 인용에 "미확인" 딱지 → ②제목 그대로 검색(구글 스칼라·RISS·DBpia) → ③원문 대조 → ④못 찾으면 삭제 → ⑤확인된 것만 출처 표기
  • 학생에게도 가르치기: 과제 조건에 "AI 활용 시 모든 인용의 원문 확인 여부 표기" 추가
  • 📦 산출물: 피드백 템플릿 + 출처 검증 체크리스트
실습 — Claude에 그대로 붙여넣기
실습 1표본 피드백 → 템플릿 확장
[역할] 너는 좋은 피드백의 구조를 분석하는 교육 코치야.
[맥락] 아래는 내가 학생 A(익명 처리)의 에세이에 정성 들여 쓴 피드백이야. 이 품질을 유지하면서 30명 전체에게 확장하고 싶어.
[요청] 이 피드백의 구조를 분석해서 다른 학생에게도 쓸 수 있는 템플릿으로 만들어줘.
[형식] 템플릿(학생마다 바뀌는 부분은 [빈칸], 유지되는 틀은 그대로) + 사용법 3줄.
[예시] 템플릿 문장 예: "[구체적 인용]에서 [강점]이 돋보입니다. 다음에는 [방법]을 시도해 보세요."
[조건] 내 말투를 유지하고, 마지막 격려 문장 자리는 "직접 작성" 표시로 비워둬.

실습 2가짜 인용 사냥 훈련
[역할] 너는 참고문헌의 실존 여부를 검증하는 학술 사서야.
[맥락] AI가 만들어 준 자료에 참고문헌이 딸려 왔는데, 인문사회 글쓰기에서 가짜 인용은 치명적이라 하나씩 확인하려고 해.
[요청] 웹에서 검색해서 다음 참고문헌들이 실제로 존재하는지 확인해줘: [참고문헌 목록]
[형식] "문헌 / 실존 여부 / 확인 출처 링크 / 저자·연도·제목 정확성" 표.
[예시] 판정 예: "○○(2021) — 검색 결과 없음, 삭제 권장 / △△(2019) — 실존, 단 연도는 2018이 맞음".
[조건] 확인이 안 되는 문헌은 "삭제 권장"으로 표시하고, 최종 판단은 내가 원문을 열어 확인할 것까지 안내해줘.
실습 3웹 검색으로 근거 자료 수집·검증
[역할] 너는 출처를 반드시 남기는 꼼꼼한 학술 리서처야.
[맥락] [과목명]의 서술형 평가와 피드백에 쓸 최신 근거 자료(사례·통계·연구)가 필요해. 출처가 불분명한 자료는 쓸 수 없어.
[요청] 웹에서 검색해서 [주제]에 대한 최근 2년 이내 근거 자료를 3개 찾아줘.
[형식] 자료마다 (1) 한 줄 요약 (2) 출처(기관·발행일·링크) (3) 평가·피드백에서 활용하는 방법 (4) 내가 원문에서 직접 확인할 항목을 표로.
[예시] 활용 방법 예: "서술 문항의 제시 자료로 — 학생이 이 통계를 해석해 논증하게 함".
[조건] 출처가 확인되지 않는 자료는 아예 제외하고, 자료가 3개 미만이면 검색어를 바꿔 다시 시도한 과정도 알려줘.
실습 4학생 유형별 피드백 변주
[역할] 너는 학생의 마음 상태까지 고려하는 상담 감각의 교수야.
[맥락] 같은 피드백 템플릿도 학생 상태에 따라 다르게 전달돼야 해.
[요청] 내 피드백 템플릿을 세 학생 유형에 맞게 변주해줘. (1) 잘하지만 안주하는 학생 — 도전 과제 제시형 (2) 노력하지만 방법을 모르는 학생 — 구체적 방법 안내형 (3) 자신감을 잃은 학생 — 강점 재발견형.
[형식] 유형별 예시 피드백 각 1편(5줄 이내).
[예시] 강점 재발견형 문장 예: "이 문단의 비유는 반에서 가장 신선했습니다 — 이 감각을 본론에도 써 보세요."
[조건] 세 편 모두 같은 과제 상황을 가정해서 차이가 잘 보이게.
